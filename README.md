## How to set up

- Run ````gulp```` in root folder
- Run ````npm run dev```` to serve on localhost:3000

## To build for production and deployment

- Run ````npm run build````
- Run ````npm run start```` to serve on localhost:3000 with build files
- Run ````now```` to begin deployment
- Run ````now -alias <given url johnnyqbui.com>````
